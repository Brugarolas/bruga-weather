# bruga-weather

> Weather app made with React, Redux and React Router

> [Try it here!](https://brugarolas.github.io/bruga-weather/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build
```
